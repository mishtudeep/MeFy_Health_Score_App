# health-score-app

This application calculates health score for a person. <br>
It calculates the score based on 8 parameters: <br>
<br>
    1. BMI <br>
    2. HEART RATE <br>
    3. ALCOHOL DAILY CONSUMPTION <br>
    4. ALCOHOL WEEKLY CONSUMPTION <br>
    5. BLOOD PRESSURE <br>
    6. BLOOD SUGAR <br>
    7. CHOLESTEROL <br>
    8. SMOKING <br>
